Escape from Tarkov


Stay In Tarkov

The largest ever mod that enabled Coop for Escape from Tarkov offline. This mod was written from scatch using a combination .NET, Typescript, Node & Reverse Engineering.

This mod had my heart and soul poured into it and after 2 years achieved it's goal at succesfully replicating Live experience.

Learned a tremendous amount about Open Source Project Management, Branching, GitHub Actions, Unity Game Engine and Game Networking.

This project is open source and actively forked by other groups. I left it on GitHub under the leadership of other great developers
