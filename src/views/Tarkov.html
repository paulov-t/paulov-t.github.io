Escape from Tarkov

Escape from Tarkov is a hardcore and realistic online first-person shooter with RPG elements and a story-driven walkthrough. The game is set in the fictional Norvinsk region, where private military companies are fighting for control over the area. Players can engage in raids, complete quests, and interact with various factions while managing their character's health, equipment, and inventory.

I was known as Paulov-t in the Escape from Tarkov modding community, where I developed the "multiplayer" of "SPT" when I created the project named Stay in Tarkov.

Note: I am no longer part of the project. I now play official Tarkov PvP and PvE.

Projects:

  • Paulov.Bepinex.Framework - A framework for BepInEx mods.
  • Paulov.Unity.BepInEx.Networking - A networking library for BepInEx mods that provides a simple and efficient way to handle network communication in Unity games.
  • Paulov.Tarkov.Minimal - A minimalistic mod for Escape from Tarkov that provides basic functionality.
  • Paulov.Tarkov.Local - A mod that allows you to play Escape from Tarkov offline with fixes to local games.
  • Paulov.Tarkov.Launcher.WPF - A WPF-based launcher for Escape from Tarkov that provides a user-friendly interface for managing mods and settings.
  • Paulov.Tarkov.LogRedirector - A mod that redirects Escape from Tarkov logs to the BepInEx log system, allowing for better log management and analysis.

The largest ever mod that enabled Coop for Escape from Tarkov offline ("SPT"). This mod was written from scatch using a combination .NET, Typescript, Node & Reverse Engineering.

This mod had my heart and soul poured into it and after 2 years achieved it's goal at succesfully replicating Live experience and created a framework for other groups to use.

I learned a tremendous amount about Open Source Project Management, Branching, GitHub Actions, Unity Game Engine and Game Networking.

This project is open source and actively forked by other groups such as Project Fika.
